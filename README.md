## Usage

```bash
# clone this repository
$ git clone https://github.com/hacker-home/Photo-Module.git

# Install dependencies
$ npm install

# Seed database
$ npm run seed

# Bundle files with webpack
$ npm run react-dev

# Run the app
$ npm run start
```
